## ✨ 功能全景

### 🪄 核心特性概览
- **🚀 双点即达页首** - 在页面右侧区域**双击** 即可平滑滚动至页面顶部 免去手动滚动的繁琐
- **🔍 跨引擎即刻搜** - 预设 Google / Bing / Baidu 主流搜索引擎跳转按钮 可通过 `Ctrl/Alt + Z/X/C` 快捷键 将当前搜索词**一键传送**至其他搜索引擎
- **💡 聚焦搜索文本** - 开启后 搜索结果的关键词将以柔和的**高亮效果**突出显示 帮助快速定位关键信息 通过 `Ctrl/Alt + H` 或点击工具栏按钮切换
- **📖 分栏结果视图** - 网页搜索结果支持**双栏模式** 信息密度与阅读舒适度同步提升 通过 `Ctrl/Alt + D` 快捷键或点击工具栏按钮切换
- **🧭 快捷类别导航** - 使用 `Ctrl/Alt + =`（下一项）和 `Ctrl/Alt + -`（上一项） 在「全部 / 图片 / 视频 / 新闻」搜索类别间**流畅切换**
- **✍️ 搜索语法助手** - 在搜索框上方预置常用的**语法快捷按钮** 内设「精确搜索 / 搜索排除 / 限定站点 / 筛选文件」按钮 点击即可将相应语法智能运用于当前或选中的搜索词
- **🔗 网链一键拷贝** - 鼠标悬停于搜索结果时 标题右侧会灵动浮现**拷贝图标** 点击即可将该结果的网址拷贝到剪贴板
- **🧰 智能持续优化** — 动态感知并适应 DuckDuckGo 的内容更新 确保各项增强功能**无缝延伸**至新载入的搜索结果 带来始终如一的浏览体验
- **💾 无缝记忆偏好** - 脚本会记住各项功能开关选择 当下一次探索 DuckDuckGo 时 这些个性化设置将**自动恢复** 确保流畅如一的使用感受

### ⌨️ 快捷按键指南
| ⌨️ 快捷按键 | ⚡️ 功能描述 |
| :-: | :-: |
| `Control / Alt + Z` | 转至**谷歌**搜索 |
| `Control / Alt + X` | 转至**必应**搜索 |
| `Control / Alt + C` | 转至**百度**搜索 |
| `Control / Alt + H` | 开关**聚焦搜索文本** |
| `Control / Alt + D` | 开关**分栏结果视图** |
| `Control / Alt + =` | 切换至**下一个**搜索类别 |
| `Control / Alt + -` | 切换至**上一个**搜索类别 |
